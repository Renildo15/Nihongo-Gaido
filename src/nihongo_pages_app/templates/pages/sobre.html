Nihongo Gaido

Nihongo gaido é um sistema simples com o intuito de ajudar aqueles que estão estudando japonês. Atualmente o sistema não tem muitas features avançadas pois etá na sua primeira versão, mas consegue dar conta do recado. O sistema foi desenvolvido por apenas uma pessoa utilizando Django e python. Tendo em vista que o desenvolvimento desse sistema foi mais para uso pessoal do que comercial, ele se torna simples. Futuramente o desenvolvedor pensa em melhorar o sistema e quem sabe, tenta a sorte no mercado.

Para que serve o Nihongo Gaido?

O sistema tem como objetivo ajudar nos estudo da língua japonesa. Onde futuramente o sistema possa ter uma atualização se adaptando para demais idiomas desejados pelo os usuários.
